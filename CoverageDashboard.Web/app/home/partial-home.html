Capability Dashboard

  • 1.0

    Cabability Dashboard is the access to the knowledge coverage related to a project at one place.

    • The main entities(entries) to the application are

      Project info.

      Employee skill matrix specific to the Project.

      Skills (domain, technology, QA and Build)

      Resource Skill mapping

    • Capability Dashboard can help to

      Track Resource progress based on the mappings.

      Identify Training requirement (External/OJT(on-job training)).

      Dashboard for summary info to know the project status and help business planning.

      Used to make decisions during ramp-up/down.
